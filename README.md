# Edu Map

Edu Map — это веб-сервис для **визуализации учебных планов** в виде интерактивного графа курсов.
Платформа позволяет студентам наглядно видеть структуру образовательной программы, понимать последовательность изучения дисциплин и планировать свою траекторию обучения.

## Технологии

Проект реализован с использованием современных веб-технологий:

- React
- TypeScript
- Vite
- ESLint
- Prettier
- Stylelint

## Установка и запуск

Клонировать репозиторий:

```bash
git clone https://github.com/visualizer-tech-project/frontend.git
cd frontend
```

Установить зависимости:

```bash
npm install
```

Запустить проект в режиме разработки:

```bash
npm run dev
```

## Сборка проекта

```bash
npm run build
```

## Проверка кода

Проверка JavaScript / TypeScript:

```bash
npm run lint
```

Проверка CSS:

```bash
npm run lint:css
```

Форматирование кода:

```bash
npm run format
```

## Структура веток

В проекте используется следующая модель ветвления:

```
main — стабильная версия проекта
dev — основная ветка разработки
```

## Для openapi

Вам необходимо вставить openapi.yml в корень проекта и в случае изменения openapi можно перегенерировать типы, используя команду ниже:

```
	npx openapi-ts
```
